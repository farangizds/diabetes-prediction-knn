# Diabetes Prediction using KNN

This project builds a machine learning model to predict diabetes based on medical indicators using the K-Nearest Neighbors algorithm.

## Project Steps

- Data cleaning
- Handling missing values
- Exploratory Data Analysis (EDA)
- Feature scaling
- Model training
- Hyperparameter tuning
- Model evaluation

## Model Performance

Accuracy: ~73%

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Seaborn
- Matplotlib
